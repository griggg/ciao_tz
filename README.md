# ciao_tz
Для запуска необходимо установить numpy, matplotlit (и другие библиотеки).

Нужно установить python acados https://docs.acados.org/python_interface/index.html 
Скопировать репозиторий, собрать cmake, установить переменные окружения, установить в pip
